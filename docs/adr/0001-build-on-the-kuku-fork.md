# Build on the Kuku Fork

Kuku OS will be implemented as a fork of `kuku-mom/kuku`, not as a separate Mac app. The core user desire is to preserve Kuku's Markdown vault, notes, editor, search, graph, backlinks, and existing AI foundation while adding a Today Dashboard and operating layer. This keeps upstream Kuku behavior visible and reduces the risk of rebuilding a note app beside the note app the user already wants to keep.
